Avaliação Web e Servidores

O que é uma publicação web?

Uma publicação é conteúdo disponibilizado online, acessível ao público . Isso inclui artigos em blogs, vídeos, imagens e outros tipos de mídia.

Quais tipos de Aplicações Web existem?

  • Aplicações Web Estáticas: São compostos com painas de conteudos fixo, e não precisão de progração(sem backend) tendo apenas HTML e CSS.
  • Aplicações Web Dinâmicas: Já esse tem interação com usuario, geralmente contem banco de dados, uzando linguagem de programação.

    • O que é um domínio e um DNS?

    Um domínio é o responsavel por permitir que os usuários acessem a WEB por meio de nomes, ex:www.everton.com" . O DNS (Domain Name System) é o sistema que traduz esses nomes de domínio em endereços IP, permitindo que os navegadores carreguem os recursos da internet.

    Servidor Compartilhado vs. Servidor Dedicado

    Um servidor compartilhado hospeda vários sites em um único servidor físico, compartilhando recursos como CPU e memória. Já um servidor dedicado oferece todos os recursos de um servidor exclusivamente para um único cliente.

    Vantagens e Desvantagens

    • Servidor Compartilhado: É um servidor fisico que compartilha varios sites, tendo seus recursos utilizado por varios sites hospedados nele.
    • Servidor Dedicado: Servidor utilizado por apenas uma instituição, permitindo controle total sobre as suas aplicações.

    O que é um servidor web?

    Um servidor web é um software e hardware que utiliza o protocolo HTTP para responder a solicitações de clientes, entregando páginas web ao navegador. Os dois principais servidores web são o Apache e o NGINX.

    Principais Tipos de Servidores Web

    Apache HTTP Server (Apache)

    : É um dos servidores web mais populares e amplamente utilizados. É de código aberto e pode ser executado em diversos sistemas operacionais, como Linux, Windows e macOS. Características: Flexível, com suporte a módulos que permitem adicionar funcionalidades extras, como autenticação e criptografia2.

    NGINX

    Descrição: Conhecido por sua alta performance e capacidade de lidar com um grande número de conexões simultâneas. Também é de código aberto e pode funcionar como servidor web, proxy reverso e balanceador de carga. Características: Eficiência no uso de recursos, ideal para sites de alto tráfego.

    Como fazer o backup de um projeto de forma compactada

    Para fazer o backup de um projeto, você pode usar ferramentas como o tar no Linux. Aqui está um exemplo de como fazer isso:

    tar -czvf backup_projeto.tar.gz /caminho/do/projeto

    Para restaurar o backup, use o comando:

    tar -xzvf backup_projeto.tar.gz -C /caminho/de/destino

    O que é um servidor de banco de dados?

    Um servidor de banco de dados é um sistema que fornece serviços de banco de dados aos clientes. Ele gerencia o acesso e a recuperação de dados. Exemplos incluem MySQL e PostgreSQL.

    Backup e Restauração

    Para fazer o backup de um banco de dados MySQL, use o comando:

    mysqldump -u usuário -p banco_de_dados > backup.sql

    Para restaurar o backup, use:

    mysql -u usuário -p banco_de_dados < backup.sql

    O que é um serviço SSH?

    O SSH (Secure Shell) é um protocolo que permite acessar e gerenciar servidores remotamente de forma segura. Ele é usado para transferir arquivos e executar comandos em servidores remotos.

    O que é Linux?

    O Linux é um sistema operacional open source baseado no kernel Linux. Ele é amplamente utilizado em servidores web devido à sua estabilidade e segurança.

    Principais Distribuições Usadas como Servidor Web

    • Ubuntu Server: Popular e fácil de usar.
    • CentOS: Conhecido por sua estabilidade.
    • Debian: Famoso por sua segurança e robustez.

    Comandos Básicos do Linux

    • ls: Lista arquivos e diretórios.
    • cd: Navega entre diretórios.
    • cp: Copia arquivos ou diretórios.
    • mv: Move ou renomeia arquivos ou diretórios.
    • rm: Remove arquivos ou diretórios.
    • chmod: Altera permissões de arquivos ou diretórios.
